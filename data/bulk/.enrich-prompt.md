Write a Node.js script at /workspace/recommender/data/bulk/enrich.js that enriches shows with Wikipedia data.

## What it does
1. Reads shows from PostgreSQL (SELECT id, title, type, year FROM shows WHERE id NOT IN (SELECT show_id FROM show_details) ORDER BY id)
2. For each show, searches Wikipedia API: https://en.wikipedia.org/w/api.php?action=query&list=search&srsearch=TITLE&format=json
3. Takes the first result, fetches the page extract: https://en.wikipedia.org/w/api.php?action=query&prop=extracts&exintro=1&explaintext=1&pageids=PAGEID&format=json
4. Also fetches categories: https://en.wikipedia.org/w/api.php?action=parse&pageid=PAGEID&prop=categories&format=json
5. For cast/director, parses infobox from: https://en.wikipedia.org/w/api.php?action=parse&pageid=PAGEID&prop=text&section=0&format=json
   - Look for "Directed by", "Starring", "Created by", "Written by" in the infobox HTML
6. Stores results in show_details, show_people, show_categories tables
7. Exports enriched data to /workspace/recommender/data/bulk/enriched.sql.gz

## Requirements
- Use node-fetch or built-in fetch (Node 22 has it)
- Use the pg pool from server/src/db.js (import)
- Process in batches of 20, sleep 2 seconds between batches (Wikipedia rate limit)
- Log progress: Show N/TITLE: synopsis_len chars, N categories, N people
- Handle missing Wikipedia pages gracefully (store NULL, continue)
- Run with: node data/bulk/enrich.js
- Export final SQL to data/bulk/enriched.sql.gz
